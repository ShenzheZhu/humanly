# Humanly Paper Materials

This branch is intentionally paper-only. It is used for version-controlling
planning notes, prompts, study instruments, detector outputs, analysis tables,
figures, and other materials for the Humanly EMNLP 2026 System Demonstrations
submission.

The app source code lives on `main` and related product branches. Do not merge
application code into this branch.

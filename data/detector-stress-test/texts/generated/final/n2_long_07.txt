The paper proposes a memory-augmented attention architecture for video captioning. Its main idea is to improve standard attention-based sequence-to-sequence models by keeping a recurrent memory of earlier attention decisions while generating a sentence. The model has three parts: a temporal encoder over video frames using convolutional feature maps and location attention, a hierarchical attention and memory module that attends over the encoded frame sequence conditioned on the decoder and memory states, and a recurrent decoder that predicts the next word. The authors evaluate the method on MSVD and Charades and report competitive results, including strong BLEU scores on MSVD without optical flow, C3D features, or fine-tuned action features.

The submission addresses a real weakness in simple video captioning models. A good caption often requires attention to an event that unfolds over time, not just to a single frame or a short local segment, while also keeping track of what has already been described. The proposed architecture is a plausible way to approach this. The separation between frame-level temporal encoding, spatial or location attention within convolutional maps, and a higher-level memory over generated words and attended visual states is sensible. Using convolutional feature maps rather than only fully connected CNN features lets the model attend spatially, and the recurrent memory module could in principle reduce repeated or inconsistent attention during decoding.

Another strength is that the model is trained end to end and does not rely on hand-engineered semantic representations or explicit action labels. That is desirable for captioning, where the goal is to learn the visual-to-language mapping directly. The paper also evaluates on two datasets with different properties. MSVD is a common benchmark with many reference captions, while Charades is more challenging and action-centered, with fewer captions per video. Including Charades is useful because it tests whether the model can handle more realistic activities and less dense language supervision. The qualitative examples also help show where the method succeeds and where it fails.

I have several significant concerns, however, about the technical clarity, novelty, and empirical support for the claims. The first is that the proposed memory mechanism is not explained precisely enough to make its contribution clear. The HAM module is described as storing previous attention and an encoded version of the input videos conditioned on previously generated words. The equations, though, look like a fairly standard recurrent attention mechanism: attention scores are computed from the encoded video states, the previous decoder state, and the previous memory state, and the weighted visual feature is then fed into an LSTM memory. It is not clear in what sense all previous attention distributions are explicitly stored, since the memory seems to be a compressed recurrent hidden state rather than an addressable memory. The paper repeatedly contrasts this approach with “local” attention and says the model can consider the whole video while generating each word, but conventional soft attention over all encoder states already does that. The added recurrent memory may still help, but the conceptual distinction needs to be sharper.

The novelty relative to existing attention and reviewer-style captioning models is also not fully convincing. Conditioning attention on the previous decoder state and earlier contextual information is common in sequence-to-sequence attention models. The paper argues that prior work does not explicitly store all previous attention, but the implementation does not appear to store the full sequence of past attention maps either. It summarizes prior decisions through an LSTM state. This may be a reasonable design, but the paper should be more careful when presenting it as a qualitatively different memory architecture. A stronger discussion would explain whether the benefit comes from adding another recurrent layer between attention and decoding, using convolutional feature maps with spatial attention, conditioning attention on a memory state, or some combination of these factors.

The experiments do not adequately isolate these possibilities. The ablation table is useful, but the results are somewhat confusing and do not support all of the claims. On MSVD, the full HAM+TEM model has the best METEOR among the ablations, but HAM without TEM has better BLEU-2, BLEU-3, and BLEU-4, along with similar CIDEr. The simpler attention model has better CIDEr than the full model. The claim that TEM and HAM complement each other is therefore only partly supported. If the full model is better, that should be shown more consistently across metrics or explained as a tradeoff between metrics. It would also be important to report variance over multiple runs, since the absolute differences are small and could plausibly come from random initialization, frame sampling, or hyperparameter search.

There are also inconsistencies in the experimental setup that make the results harder to interpret. The ablation study uses 40 frames per video, while the main MSVD comparison uses 8 frames per video. This makes it difficult to understand how frame sampling affects the architecture, especially because the temporal encoder is one of the proposed contributions. If temporal modeling is important, the paper should analyze performance as a function of the number of sampled frames. It should also clarify whether hyperparameters were tuned separately for each ablation and baseline, or whether the full model received more tuning. Because the reported hidden sizes are quite large and selected by random search, the fairness of these comparisons matters.

The state-of-the-art claim is overstated. On MSVD, the model has strong BLEU scores among methods that do not use external motion or action features, but it is not best on all metrics. Another method in the table has substantially higher METEOR, and methods using additional features outperform the proposed model on BLEU-4 and METEOR. It would be fair to say that the method is competitive and achieves the best reported BLEU-4 among the listed methods without external features. The broader claim in the abstract and conclusion that it outperforms all previously proposed methods and reaches a new state of the art is not supported by the tables. The Charades results improve on the dataset baseline, but the gain is modest, and the absolute scores remain far below human performance. More analysis is needed before drawing strong conclusions about action understanding.

The paper also lacks several implementation details needed for reproducibility. The frame sampling strategy is not fully specified. The decoding procedure is missing important information, including whether greedy decoding or beam search is used, what beam size is selected, how unknown words are handled, and how maximum caption length is chosen. The preprocessing description mentions NLTK tokenization, but does not describe vocabulary thresholds, handling of rare words, start and end tokens, or whether captions are lowercased. The architecture section gives equations for the main modules, but some dimensions and definitions are unclear, especially in the attention update where repeated decoder and memory states are concatenated or broadcast across frames. The loss function omits the decoder parameters from the listed parameter set. These issues may not be fatal, but they reduce confidence in the empirical results.

The qualitative examples point to another limitation: the model can produce fluent but semantically wrong captions, sometimes hallucinating common objects or actions. This is a known problem in captioning systems, but it matters here because the paper claims that memory-augmented attention improves reasoning over actions and temporal structure. Some incorrect examples suggest that the model may rely heavily on language priors. A stronger evaluation would include attention visualizations over time and space, showing whether the model attends to action-relevant frames when generating verbs and object-relevant regions when generating nouns. Since the central contribution is an attention and memory mechanism, visualizing or quantitatively evaluating its behavior would be valuable.

I have several questions for the authors. In what precise sense does the model store all previous attention decisions, rather than only a recurrent summary of them? Have you compared HAM with a standard attention decoder that has an additional recurrent layer and a comparable parameter count, to rule out the possibility that the gains come from increased capacity? How sensitive are the results to the number of sampled frames, and why are 40 frames used in the ablation while 8 or 16 frames are used in the main experiments? Were the ablation models given the same hyperparameter search budget as the full model? What decoding method and beam size were used for the reported results? Can the authors provide attention visualizations or an analysis showing that the memory state reduces repeated attention or improves verb generation? Can the state-of-the-art claims be revised to reflect the mixed results across metrics and feature settings?

Overall, the direction is promising and the architecture is reasonable, but the submission does not yet meet the level of clarity and empirical rigor I would expect for acceptance. Conditioning video attention on a recurrent memory of prior attended context is a worthwhile idea, and the results are competitive. Still, the novelty over standard recurrent attention mechanisms is not clearly established, the ablations do not convincingly validate the claimed contributions, and some claims are stronger than the evidence supports. My recommendation is rejection in its current form, with encouragement to resubmit after clarifying the memory mechanism, strengthening the baselines and ablations, adding reproducible experimental details, and tempering the performance claims.
